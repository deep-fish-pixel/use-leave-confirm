# Nio App Web
本项目通过[nio-app](https://pkg.nioint.com/web/detail/@nio-fe/nio-app)创建，[点击](https://pkg.nioint.com/web/detail/@nio-fe/nio-app)可查看生成更多功能的配置

### Cmd

启动
```
npm run start
```

构建
```
npm run build
```

eslint检查
```
npm run lint:eslint
```

eslint格式化
```
npm run lint:prettier
```



### Main modules
* vite + vue3 + typescript + jsx
* pinia状态管理
* css预处理工具: scss
* nio编码规范
* git提交规范：husky


* typescript mock
* 扩展keep-alive
* 请求进度条
* 权限指令
* 组件库：element-ui
